AI-Powered Video Processing Backend

Features:

  • Upload and process video files
  • Generate viral clips using AI analysis
  • Automatic subtitle generation with Whisper
  • Format conversion (including vertical 9:16 format)
  • OpenAI integration for smart content analysis
  • FFmpeg-powered video processing

API Endpoints:

Videos:

POST /api/videos/upload - Upload a video

GET /api/videos/:videoId - Get video info

DELETE /api/videos/:videoId - Delete a video

Clips:

POST /api/clips/generate - Generate viral clips

GET /api/clips/status/:jobId - Get job status

GET /api/clips/:clipId - Download clip

GET /api/clips - List clips

Health:

GET /health - Server health check
